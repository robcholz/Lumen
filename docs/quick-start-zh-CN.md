# 🔥快速开始

本指南的目标是：
**在不需要任何本地开发环境的情况下，让 Lumen 成功运行起来。**

你不需要了解嵌入式系统的原理，也不需要修改任何代码。

---

> ⚠️ 在开始之前
>
> 本快速开始假设你**已经拥有一块 Lumen 硬件**。
>
> 如果你还没有硬件，可以直接通过 **嘉立创** 完成制作：
>
> - PCB：上传 Gerber 文件即可下单 SMT
> - 外壳：下载 STL 文件进行 3D 打印
>
> 👉 所有下单文件已准备好，见：  
> **/hardware**

### 硬件制作步骤（嘉立创）

#### 1. PCB 打样与 SMT

1. 打开嘉立创下单页面
2. 上传仓库中提供的 **Gerber(gerber.zip)文件**
3. PCB厚度1.2mm，铜厚1-2oz，双层板，黑色
4. 选择 **SMT 贴片**
5. 上传对应的 **BOM(hardware/bom.csv)** 与 **Pick & Place(hardware/pickandplace.csv)** 文件
6. 默认工艺参数即可下单

#### 2. 外壳 3D 打印

1. 下载仓库中提供的 **STL 文件**
2. 使用任意 3D 打印服务或本地打印
3. 推荐材料：PLA / PETG，黑色耗材，0.10mm精度
4. 默认打印参数即可正常装配

#### 3. 面板制造

1. 使用1/16英寸透明亚克力板
2. 切割并**双面雕刻**，雕刻深度在`top-cover.ai`中有写出
3. 对于导管柱（`light-guide.ai`），使用砂纸打磨四个角

#### 4. 显示屏

淘宝搜索1.54寸tft显示屏 240x240分辨率 SPI接口ST7789. 大小 32x43.7x2.58mm

#### 3. 组装

1. 焊接PCB到显示屏的线，使用细线，注意方向。方向应和渲染图中的方向一致
2. 反向安装PCB到`bottom-shell.stl`，然后使用502或者M1.3攻丝螺丝安装`side-shell.stl`
3. 安装导光柱和面板，使用502粘合

## 你需要准备什么

- 一块 Lumen 硬件
- 一根 USB-C 数据线
- 一台支持 Chrome / Edge 的电脑

## 第 1 步：连接设备

使用 USB-C 线将 Lumen 连接到电脑。

首次连接时，系统可能会提示这是一个新的 USB 设备，这是正常的。

## 第 2 步：打开 Web 烧录页面

打开以下页面：

👉 [**Web Flasher**](https://web.esphome.io/)

这是一个基于浏览器的烧录工具，
不需要安装任何软件。

## 第 3 步：烧录固件

1. 点击 **Connect / 连接设备**
2. 在弹出的窗口中选择 Lumen
3. 找到Release中最新的`lumen_single_bin_{version}.bin`并下载
4. 选择下载好的固件点击 **Flash / 烧录**

整个过程通常只需要几十秒。

## 第 4 步：等待设备重启

烧录完成后，Lumen 会自动重启。

如果一切正常，你将看到：

- 屏幕点亮
- UI 正常显示
- 可以通过旋钮或交互进行操作

🎉 **至此，复刻完成。**
